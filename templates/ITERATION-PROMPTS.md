# Iteration Prompts

Copy-paste snippets for Module 4. Attach screenshots or error text where noted.

---

## Fix a terminal error

```
The app crashed / shows an error. Here is the full terminal output:

[PASTE ENTIRE ERROR]

Fix this. Explain what caused it in one sentence.
```

---

## Fix UI from screenshot

_Attach screenshot first._

```
This is how the app looks on my phone right now.

Problems:
- [e.g. Title is cut off at the top]
- [e.g. Button too small to tap]

Fix these. Use [e.g. 24px] padding below the header if needed.
```
